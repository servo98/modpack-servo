# Capitulo 1: Primeras Raices

> Stage: `servo_ch1` (default — mundo nuevo)
> Trigger para Ch2: Derrotar al Guardian del Bosque + Completar entrega Ch1
> Mecanicas: [Progression](../mechanics/progression.md), [Cooking](../mechanics/cooking.md), [Dungeons](../mechanics/dungeons.md), [Champions](../mechanics/champions.md), [Bosses](../mechanics/bosses.md), [Gacha](../mechanics/gacha.md), [Game Loop](../mechanics/game-loop.md)

## Tema

Capitulo introductorio. Survival vanilla + Farmer's Delight. Aprender los sistemas core del modpack: cocinar, farmear, explorar, dungeons basicas, y el Space Elevator. Sin Create, sin clases RPG — solo fundamentos.

## Mods activos

Vanilla, Farmer's Delight (basico), FTB Quests (Recetario), Lootr, Gacha Machine, Waystones, Xaero's Minimap/Worldmap, Jade, Sophisticated Backpacks, Carry On, Spell Engine (T0 limitado), Runes (basico), Jewelry (introductorio)

## Quests (~50)

### Historia/Tutorial (12)
1. Bienvenida al mundo — tutorial basico de survival
2. Construir refugio
3. Conocer el Cooking Pot (FD)
4. Conocer la Cutting Board (FD)
5. Conocer el Stove (FD)
6. Tutorial: Space Elevator — como funciona el Terminal de Entrega
7. Tutorial: insertar items en el Terminal (click derecho directo)
8. Tutorial: Gacha Machine (quest reward: obtener la maquina)
9. Tutorial: Dungeon y llaves (quest reward: 1ra Llave Basica gratis)
10. Tutorial: Champions — que son, como se reconocen
11. Tutorial: Spell Binding Table (quest reward)
12. Tutorial: Rune Pouch — equipar y usar runas

### Cocina (6)
1. Cocinar primera comida FD (Cooking Pot)
2. Hacer 3 recetas diferentes en el Cooking Pot
3. Usar la Cutting Board para cortar ingredientes
4. Cocinar algo con el Stove
5. Preparar algo con el Skillet
6. Comer 5 comidas diferentes (Recetario: +1 corazon)

### Farming (6)
1. Plantar tomate (FD)
2. Plantar cebolla (FD)
3. Plantar repollo (FD)
4. Plantar arroz (FD)
5. Tener granja con 4+ crops diferentes
6. Domesticar animales (vaca, gallina, cerdo)

### Dungeon (5)
1. Construir Pedestal de Dungeon
2. Usar Llave Basica — entrar a primera dungeon
3. Sobrevivir una dungeon completa (salir vivo)
4. Obtener loot de cofre de dungeon
5. Derrotar un champion en dungeon

### Combate/RPG (5)
1. Craftear un arma T0 (Flint Dagger o Stone weapon)
2. Derrotar un champion en overworld
3. Equipar Small Rune Pouch
4. Preparar Boss Key Ch1
5. Derrotar al Guardian del Bosque

### Exploracion (4)
1. Explorar 3 biomas Terralith
2. Colocar primer Waystone
3. Encontrar un village
4. Pescar 3 tipos de pez diferentes

### Construccion (4)
1. Construir casa basica (4 paredes + techo)
2. Construir cocina (colocar workstations FD)
3. Construir almacen (10+ cofres organizados)
4. Construir granja cercada

### Coleccion (5)
1. Obtener 5 Pepe Coins
2. Hacer primer pull de gacha
3. Completar 5 comidas unicas en el Recetario
4. Equipar Copper o Iron Ring
5. Completar entrega al Space Elevator Ch1
