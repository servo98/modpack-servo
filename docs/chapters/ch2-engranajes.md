> Stage: `servo_ch2` (requiere: Boss Ch1 + Delivery Ch1)
> Trigger para Ch3: Derrotar a la Bestia Glotona + Completar entrega Ch2
> Mecanicas: [Cooking](../mechanics/cooking.md), [RPG Classes](../mechanics/rpg-classes.md), [Create Automation](../mechanics/create-automation.md), [Storage](../mechanics/storage.md), [Jewelry](../mechanics/jewelry.md)

## Tema

Primer contacto con Create (belts mueven cosas). Cocina expandida con Croptopia y fermentacion. Elegir primera clase melee (Rogue o Warrior). Storage Drawers para organizar bulk. El jugador pasa de "sobrevivir" a "construir sistemas".

## Mods nuevos

Croptopia (12 crops), Brewin' And Chewin' (Keg), Create basico (andesite tier), Storage Drawers, Macaw's Furniture basico, Rogues & Warriors (class books), Jewelry (gemas basicas)

## Quests (~45)

### Historia/Tutorial (10)
1. Introduccion a Croptopia — nuevos crops disponibles
2. Tutorial: Keg (fermentacion — quesos y bebidas)
3. Tutorial: Storage Drawers (bulk storage visual)
4. Tutorial: Create — Hand Crank y primer shaft
5. Tutorial: Create — Water Wheel (primera energia real)
6. Tutorial: Create — belts y funnels (mover items automatico)
7. Tutorial: Class books — elegir Rogue o Warrior
8. Tutorial: Arms Station — craftear gear de clase
9. Tutorial: Jewelry — gemas y anillos
10. Explicar Citrine/Jade ores y donde encontrarlos

### Cocina (5)
1. Hacer queso en el Keg
2. Fermentar primera bebida alcoholica (cerveza o hidromiel)
3. Hacer una receta con ingredientes Croptopia
4. Comer 15 comidas diferentes (Recetario: +2 corazones)
5. Producir 3 tipos de bebidas fermentadas

### Farming (6)
1. Plantar 4 crops nuevos de Croptopia
2. Plantar 8 crops nuevos
3. Plantar los 12 crops Croptopia nuevos
4. Expandir la granja (20+ plots)
5. Granja de uvas (para Keg)
6. Granja de soybeans o peanuts

### Dungeon (4)
1. Hacer 3 dungeon runs Basicas
2. Obtener loot mejorado de dungeon
3. Obtener runas de dungeon (Fire o Healing)
4. Acumular 20 Pepe Coins de dungeons

### Combate/RPG (5)
1. Elegir clase melee (Rogue o Warrior)
2. Craftear armor T1 de clase (Arms Station)
3. Craftear arma T1 de clase
4. Equipar Fire o Healing rune
5. Derrotar a la Bestia Glotona

### Exploracion (5)
1. Encontrar ore de Citrine
2. Encontrar ore de Jade
3. Explorar 3 biomas nuevos
4. Colocar 3 Waystones
5. Encontrar village con NPCs

### Construccion (4)
1. Crear sala de Storage Drawers organizada
2. Instalar primer Water Wheel funcional
3. Conectar primer belt (mover items entre dos puntos)
4. Mejorar la cocina con Keg instalado

### Coleccion (4)
1. Hacer 5 pulls de gacha
2. Equipar primer ring de gema (Citrine o Jade)
3. Producir 32 Andesite Alloy (primer Create craft)
4. Completar entrega al Space Elevator Ch2
