3. Comer 30 comidas diferentes (FTB Quests Recetario: +3 corazones)
4. Instalar cocina funcional (nevera+estufa+fregadero)

### Farming (6)
1. Plantar 4 frutas nuevas
2. Plantar todas las frutas (10)
3. Granja de banana o mango
4. Granja de Nether Wart
5. Expandir con Expanded Delight crops
6. Automatizar harvest con Create Harvester

### Dungeon (5)
1. Craftear primera Llave Avanzada
2. Completar dungeon Avanzada
3. Obtener primera Esencia de Dungeon
4. Obtener unique jewelry (5% chance)
5. Hacer 3 runs Avanzadas

### Combate/RPG (5)
1. Elegir clase magica (Wizard, Paladin, o Priest)
2. Craftear armor T1 magica
3. Craftear Medium Rune Pouch
4. Craftear runas en Rune Crafting Altar
5. Derrotar al Coloso Mecanico

### Exploracion (5)
1. Entrar al Nether
2. Encontrar fortaleza Nether (YUNG's)
3. Obtener Blaze Rods

5. Encontrar ores Ruby/Sapphire

### Construccion (4)
1. Instalar Tom's Storage terminal
2. Construir primera linea Create

4. Instalar cocina funcional Refurbished

### Coleccion (5)
1. Obtener Ruby y Sapphire
2. Equipar ring y necklace de gema
3. Hacer 3 bebidas diferentes
4. Acumular 50 Pepe Coins
