# Capitulo 6: Maestria

> Stage: `servo_ch6` (requiere: Boss Ch5 + Delivery Ch5)
> Trigger para Ch7: Derrotar al Senor de las Cosechas + Completar entrega Ch6
> Mecanicas: [RPG Classes](../mechanics/rpg-classes.md), [Enchantments](../mechanics/enchantments.md), [Create Automation](../mechanics/create-automation.md), [Cooking](../mechanics/cooking.md)

## Tema

Optimizacion total. Menos volumen bruto, mas items dificiles de obtener. T3 Netherite upgrades para todo el equipo RPG. Enchant nivel V. Create Deco para hacer la fabrica bonita. Dragons Plus para procesamiento exotico (Ending, Freezing). Trenes full auto 24/7. Recetas multi-step que encadenan 3+ workstations. El jugador ya domina todos los sistemas — este capitulo prueba maestria.

## Mods nuevos

Create Deco (389 bloques decorativos industriales), Dragons Plus (Ending/Freezing processing, dye fluids), Mechanical Roller, Enchant nivel V, Tipped Arrows de Spell Power
