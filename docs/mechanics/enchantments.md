# Sistema de Enchantments

> Fuente: GDD v2, seccion 4.2
> Mods: Vanilla (gateado), Spell Power (enchants magicos), Create Enchantment Industry
> Relacionado: [RPG Classes](rpg-classes.md), [Progression](progression.md), [Create Automation](create-automation.md)

## Overview

Enchantments gateados por capitulo con ProgressiveStages. Enchants magicos (Spell Power) se suman en Ch5.

## Vanilla gateado por capitulo

| Cap | Max nivel de enchant |
|-----|---------------------|
| 1-2 | Nivel I-II |
| 3-4 | Nivel III |
| 5 | Nivel IV + Enchantment Industry (auto-enchanting) |
| 6+ | Nivel V, todo |

## Enchants magicos (Spell Power, Ch5+)

| Enchant | Efecto |
|---------|--------|
| Spell Power | Aumenta dano de spells |
| Sunfire | Dano de fuego magico |
| Soulfrost | Dano de hielo/soul |
| Energize | Dano de rayo |
| Spell Haste | Reduce cooldown |
| Spell Volatility | Mas crit pero menos consistente |
| Amplify Spell | AoE mas grande |
| Magic Protection | Reduce dano magico recibido |

## Pociones de Spell Power (Ch5+)

19 tipos via brewing. Cada uno boost un tipo de spell power. Se pueden hacer Tipped Arrows (Ch6+).

## Create Enchantment Industry (Ch5+)

| Bloque | Funcion |
|--------|---------|
| Disenchanter | Quita enchants de items |
| Enchanting Guide + Deployer | Auto-enchanting especifico |
| Hyper Experience | XP liquido para automatizar |

Automatiza enchants magicos para todo el equipo RPG.
