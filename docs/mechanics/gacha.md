# Sistema Gacha

> Fuente: GDD v2, seccion 3.11
> Mod: Bloo's Gacha Machine
> Relacionado: [Tokens](tokens.md), [ServoMart](servomart.md), [Accessories](accessories.md), [Dungeons](dungeons.md)
> Balance detallado: docs/balance/gacha-rates.md

## Overview

Maquina gacha como engagement hook, NO pay-to-win. Usa Pepe Coins.

## Reglas

- **NO crafteable** — se obtiene como quest reward en Ch1
- **Unica por grupo** — no se pueden hacer mas
- Usa **Pepe Coins** (ver [Tokens](tokens.md))

## Gacha de Artefactos (principal)

| Rareza | Probabilidad | Contenido |
|--------|-------------|-----------|
| Comun | 60% | Materiales utiles del capitulo actual |
| Uncommon | 25% | Accesorio T1-T2, materiales del sig capitulo |
| Raro | 10% | Accesorio T2-T3, moldes de postres raros |
| Epico | 4% | Llave de Dungeon avanzada, accesorio T3 |
| Legendario | 1% | Accesorio T4, item unico |

**Pity**: +2% epico por pull sin epico+, hard pity a **50 pulls**.

Costo: **10 Pepe Coins** por pull (~15 pulls/capitulo).

## Gacha de ServoMart (muebles)

- Separado del principal
- Caja misteriosa con mueble random
- Costo: **5 Pepe Coins** por pull (~6 muebles/capitulo)
- Ver [ServoMart](servomart.md)

## Timing

- Primera pity: ~Ch3-4
- ~150 tokens por capitulo → ~15 pulls de artefactos O 30 pulls de muebles
