# Quest System (FTB Quests)

> Fuente: GDD v2, seccion 4.3
> Mods: FTB Quests, FTB Teams, FTB Chunks
> Relacionado: [Progression](progression.md), [Tokens](tokens.md), [Space Elevator](space-elevator.md), [Game Loop](game-loop.md)

## Overview

~400 quests totales (~50 por capitulo) via FTB Quests. Guian al jugador a traves de los 8 capitulos.

## Principios

- Cada quest tiene un **objetivo concreto** — nada de "explora biomas por explorar"
- Rewards incluyen Pepe Coins (3-10 segun dificultad)
- Quest book es la guia principal del jugador
- FTB Teams: progresion compartida en multiplayer
- FTB Chunks: claim de chunks para proteccion + force-load para trenes

## Categorias de quests por capitulo

Cada capitulo tiene ~50 quests distribuidas en:

| Categoria | Quests aprox | Descripcion |
|-----------|-------------|-------------|
| Historia/Tutorial | ~12 | Mecanicas nuevas del capitulo |
| Cocina | ~8 | Recetas, workstations, variedad |
| Farming | ~6 | Crops, granja, animales |
| Dungeon | ~5 | Runs, loot, survival |
| Combate/RPG | ~5 | Clases, gear, boss prep |
| Exploracion | ~5 | Biomas, recursos, Waystones |
| Construccion | ~4 | Base, cocina, fabrica |
| Coleccion | ~5 | Comidas unicas, tokens, gacha |

## Quests especiales

- **Boss fight**: quest de matar al boss del capitulo (requiere Boss Key)
- **Space Elevator delivery**: quest de completar entrega (trigger para stage unlock)
- **Gacha Machine**: quest reward en Ch1 (unica por grupo)
- **Spell Binding Table**: quest reward en Ch1
